# Importing Student Data Instructions

## Overview
This document provides instructions for importing the student data from the spreadsheet into the ministry-hub-sync system using the Supabase dashboard.

## Prerequisites
1. Access to the Supabase project dashboard
2. Appropriate permissions to execute SQL queries

## Steps to Import Data

2. Go to the SQL Editor section

2. Verify that the `process_estudantes_batch` function exists in your database
3. Ensure that the JSON data structure matches the expected format
4. Contact the development team for assistance if needed
